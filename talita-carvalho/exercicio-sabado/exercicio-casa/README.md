## Exercícios para casa 


### 1. Revisando conceitos

Vamos revisar conceitos? Resposta as perguntas abaixo com as suas palavras: 

```
1. O que é iteração?
R: a iteração é responsável por incrementar um objeto até que uma determinada lógica seja falsa.  

2. Qual a diferença de uma function padrão e a arrow function?
R: a sintaxe entres as duas é diferentes, sendo a arrow function mais curta, enquanto que a function padrão é maior. Uma outra diferença é relacionada a forma como se trabalha com o escopo, pois a arrow function não dispõe de um this. 

3. Como funciona o método map()? 
R: o map retorna um novo Array modificado baseando-se em cada item do array. 

```
