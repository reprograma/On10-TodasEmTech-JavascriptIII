## Exercícios para casa 


### 1. Revisando conceitos

Vamos revisar conceitos? Resposta as perguntas abaixo com as suas palavras: 

```
1. O que é iteração?
É a repetição de alguns comandos que só são executados se uma certa condição é obedecida.


2. Qual a diferença de uma function padrão e a arrow function?
A forma de escrever:
Arrow function: () => {} 
Efuncton padrão: function() {}


3. Como funciona o método map()? 
O map aceita tres argumentos que são o valor atual, o index, e o array, ele é um callback que chama uma função para cada elemento do array.
Ele também retorna um novo array com o resultado, não altera o array original.
