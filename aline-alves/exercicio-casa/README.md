## Exercícios para casa 


### 1. Revisando conceitos

Vamos revisar conceitos? Resposta as perguntas abaixo com as suas palavras: 

```
1. O que é iteração?

R: É a execução de uma atividade(de um comando).

2. Qual a diferença de uma function padrão e a arrow function?

R: O arrow function é a maneira de escrever as funções de forma mais resumida, deixando o código mais simples.

3. Como funciona o método map()? 

 R: Serve para mapear os objetos,  para criar um  novo array a partir de um que já exista assim não altera o oginal, 

```
